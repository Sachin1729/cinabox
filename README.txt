cinabox

== DESCRIPTION:
  
Continuous Integration - in a Box

== FEATURES/PROBLEMS:
  
* The purpose of CI in a Box is to automate the setup of a Continuous Integration system by doing The Simplest Thing That Could Possibly Work.

== SYNOPSIS:

* Install Ubuntu Desktop 8.04
* cd ~
* wget [TODO: path to cinabox download]
* tar -zxvf cinabox-x.y.z.tar.gz
* cinabox/bootstrap_ruby.sh
* cinabox/setup_ci.rb

== REQUIREMENTS:

* Ubuntu Desktop 8.04 and an internet connection
